Jezyki programowania

  • HTML


    • Html- Hypertext markup language (Hypertextowy Jezyk Znacznikow)
    Jezyk oparty na znacznikach w ktorym opisujemy strukture strony www.

  • CSS


    • CSS- Cascade Style Sheets (Kaskadowe Arkusze Stylów)
    Jezyk w którym opisujemy wygląd strony www.

  • JS


    • JS- Java Script
    Jezyk w którym opisujemy zdarzenia na stronie ktore wykonywane są po stronie klienta(naszego komputera) (przykladowo zegar który jest na górze).

  • MYSQL


    • MYSQL- My structure querry language
    Jezyk do tworzenia i zarządzania bazami danych.

  • PHP


    • PHP- Hypertext Preprocesor
    Jezyk w którym opisujemy zdarzenia wykonywane po stronie serwera (np.:logowanie na strone za pomocą formularza logowania. To PHP rozkazuje odszukać w bazie danych uzytkownika o danym loginie i haśle. Jesli jest poprawne to oddaje uzytkownikowi napis zalogowano, a jesli nie to zwraca napis ze dane logowania są nieprawidłowe.
